Features

Jekyll is the best option out there to design and publish fast, secure and creative websites. It blows Drupal & Wordpress out of the water. We supercharge it with:

  • Jekyll+ CMS

    Get anybody to start adding content and media to your site within seconds with this professional Open Source CMS.

  • Fast builds

    No more waiting! You'll see your changes appear on your site within seconds, even with thousands of posts and images.

  • Versions

    Creating a new version of your site for development, staging or A/B testing is as easy as adding a branch on GitHub.

  • Access control

    Wanna share a private version of your site before you push it in production? Just add a password and we've got you covered!

  • Multilingual support

    Need to manage multiple languages? We've figured out the best and easiest way to do that and made as easy as a switch for you.

  • Integrations

    Add a private API or integrate with any other service in seconds. Goodbye complicated server setup or subdomain madness.

  • Search

    Wouldn't that be nice to be able to offer users a quick way to search a find all that content you wrote? Get set up in seconds.

  • Forms

    Need to let your users send you an email or send you their order info? No problemo; just add a form and point us in the right direction.

  • A/B testing

    Working on a new layout for the front page and wondering about how it will work for your user? Get an A/B test in place in 3 clicks.
